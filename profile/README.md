# kgentic

Software built with AI, shaped by humans.

kgentic (a play on *agentic*) is an umbrella organisation developing consumer-facing software, apps, tooling, and systems — both open source and proprietary — using cutting-edge AI-assisted development techniques.

## What we build

- **Consumer products** — apps and services designed for real people, not just developers
- **Developer tooling** — open-source tools that make AI-assisted workflows practical and repeatable
- **Systems** — infrastructure and platforms that power the products above

## How we build

Every line of code in this org is developed alongside AI — not as a novelty, but as a core part of the engineering process. We treat AI-assisted development as a first-class discipline: structured prompting, agentic workflows, human-in-the-loop quality gates, and rigorous testing.

We open-source the tools and techniques that make this work so others can build this way too.
